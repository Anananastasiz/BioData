# BioTIME Database Manager

Веб-приложение для управления и анализа долгосрочных данных о биоразнообразии из базы данных BioTIME.

## Описание проекта

Проект представляет собой веб-интерфейс для работы с метаданными исследований биоразнообразия из международной базы данных BioTIME. Позволяет просматривать, добавлять, редактировать и удалять информацию об исследованиях, а также проводить аналитику данных.

## Функциональность

- **Дашборд** - общая статистика по базе данных
- **Просмотр исследований** - список с фильтрацией и пагинацией
- **CRUD операции** - создание, чтение, обновление, удаление исследований
- **Детальная информация** - полные метаданные каждого исследования
- **Аналитика** - интерактивные графики и статистика
- **Управление образцами** - редактирование связанных образцов данных

## Стэк

- **Backend**: Python 3.10+, Flask
- **ORM**: SQLAlchemy
- **База данных**: SQLite
- **Frontend**: Bootstrap 5, Chart.js
- **Визуализация**: Plotly
- **Обработка данных**: Pandas
